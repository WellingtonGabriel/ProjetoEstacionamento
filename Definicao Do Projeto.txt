Definição do Projeto 

	

O Projeto tem o intuito de facilitar e organizar o funcionamento de um estacionamento de veículos, demonstrando que a tecnologia pode ser utilizada para uma melhoria do serviço, o projeto tem na sua composição o software desenvolvido, um web site para ter um acesso mais prático para aqueles que desejam contratar o o nosso software e um aplicativo mobile para os clientes do estacionamento assim sabendo algumas informações como horário de entrada/saída, vaga onde está o veículo, valor pago pelo serviço,assim passando mais confiança do serviço.
